Question:

Reverse a queue
Write a program to display the queue in reverse order. If the queue doesn't have a value print "Queue is empty".
Input Format

The input consists of a list of integers, negative value indicates the end of the queue.

Output Format

Print the reversed queue.

Sample Input:

1

2

3

4

-1

Sample Output:

Before reversing:

1 2 3 4

After reversing:

4 3 2 1

Note:

If queue is empty print the "Queue is empty".

Question:

Segregate 0's and 1's
Given a pattern 0’s and 1’s, can you move all the 1’s to the beginning and 0’s to the end?
Input Format

In an input array 0 and 1 are in an unsorted order.

Output Format

The output is sorted order of 0 and 1.

Sample Input:

010101011001

Sample Output:

111111000000

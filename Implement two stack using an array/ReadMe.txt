Question:

Implement two stack using an array
Write a program to implement two stacks using one array. Consider the size of the array as 1000.
Input Format

Get number of elements to be inserted in the 1st stack.
And then get its elements one be one.
Also, get the number of elements to be inserted in the 2nd stack.
And then, get its elements one be one.
Get number of elements to be deleted from the first stack.
Get number of elements to be deleted from the 2nd stack.
Output Format

Print the elements of the 1st stack.
Print the elements of the 2nd stack.
Print "Stack underflow. pop from stack 1 failed" message if pop operation failed while doing it in stack 1.
Print "Stack underflow. pop from stack 2 failed" message if pop operation failed while doing it in stack 2.
Print the elements of 1st stack after pop operation. Print the elements of 2nd stack after pop operation.
Sample Input

3

2 5 7

2

1 2

4

1

Sample Output

Stack 1 Elements:

7 5 2 

Stack 2 Elements:

2 1 

Stack 1 Elements:

Stack underflow. pop from stack 1 failed

Stack 2 Elements:

1

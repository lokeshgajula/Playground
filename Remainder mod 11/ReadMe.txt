Question:

Remainder mod 11
Given a string (of maximum length 1000) representing a large number, output as remainder modulo 11.

Input Format

Given input is a large number in the form of a string.

Output Format

Return the remainder modulo 11 of given input

Refer the sample output for formatting

Sample Input:

121

Sample Output:

0

Explanation:

121 mod 11 = 0

Question:

Deletion of second last element in the Linked List
Write a program to delete the second last element from the given Linked List. If deletion of the second last element is not possible, print "Deletion of second last element is not possible", otherwise print the list after deleting the second last element.
Input Format

The input consists of a list of integers, negative value indicates the end of the linked list.

Output Format

The output should be numbers in the list in separate by space

Refer the sample input & output for formatting specifications.

Sample Input

2

4

6

8

10

-21

Sample Output

2 4 6 10

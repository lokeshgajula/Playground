Question:

Remove Duplicates
Write a function to remove all duplicate characters from a given string Note: The duplicate elements are to be removed in such a way that when reading the string from left to right, the repeated element which occurs later should be removed.

Input Format

Given input is a string s

Output Format

Return a string with non-duplicate characters, i.e., if you have a string metti then output should be meti, keeping the order of characters same.

Refer the sample output for formatting

Sample Input:

goodmorning

Sample output:

godmrni

Explanation:

On removing duplicate elements from the original string goodmorning, we get godmrni.
